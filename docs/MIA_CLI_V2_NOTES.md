# MIA CLI v2 local patch

Adds missing smoke-test commands to `hermes_employee.py` and creates `data/employee/tasks.json`.

Supported commands:

- status
- character
- security
- tick
- validate
- decide
- rights
- skills
- sources
- ethical-check
- source-plan
- security-check

All actions are local-first. The CLI does not execute shell commands or call paid APIs.
